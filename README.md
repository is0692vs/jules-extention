# Jules Extension for VSCode

> "VSCode で Google Jules を使おう"

Jules Extension は Google の AI コーディングエージェント Jules を VSCode 内から直接操作できる拡張機能です．

## ステータス

🚧 **開発中** - Phase 0 MVP

## 予定機能

- 📝 VSCode からコーディングセッションを作成
- 📊 セッションの進捗をリアルタイムで監視
- 🔍 詳細なアクティビティログの表示
- 🚀 自動 PR 作成
- 📚 セッション履歴管理
