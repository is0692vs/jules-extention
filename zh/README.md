# VSCode 的 Jules 扩展

> "在 VSCode 中使用 Google Jules"

Jules 扩展程序是一个扩展，可让您直接在 VSCode 中操作 Google 的 AI 编码代理 Jules。

## 地位

🚧 **开发中** - 阶段 0 MVP

## 计划功能

- 📝 从 VSCode 创建编码会话
- 📊 实时监控会话进度
- 🔍 查看详细的活动日志
- 🚀 自动创建 PR
- 📚 会话历史管理
