# VSCode용 Jules 확장 프로그램

> "VSCode에서 Google Jules 사용"

Jules Extension은 Google의 AI 코딩 에이전트 Jules를 VSCode 내에서 직접 조작할 수 있게 해주는 확장 프로그램입니다.

🚧 **개발 중** - 0단계 MVP

## 예정 기능

- 📝 VSCode에서 코딩 세션 생성
- 📊 세션 진행 상황 실시간 모니터링
- 🔍 상세 활동 로그 보기
- 🚀 자동 PR 생성
- 📚 세션 기록 관리
